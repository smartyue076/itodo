# iTodo 本地规划器

iTodo 是一个面向个人使用的本地 Todo 与项目规划网页 Demo。它将计划视为有起止日期的项目，并以日、周、月、日历和复盘等视图帮助你安排与回顾工作。

> 数据保存在当前浏览器的本地存储中，适合个人离线使用；不提供云端同步或多人协作。

## 产品特性

- **本地登录**：输入用户名即可进入对应的本地工作区，数据按用户名在浏览器内隔离保存。
- **统一计划模型**：新建计划时指定开始、结束日期，无需人为区分日计划、周计划或月计划；计划会自动出现于日期范围覆盖到的视图中。
- **子计划**：主计划可创建一层子计划。子计划同样包含名称、描述、起止日期、任务和完成状态，且日期范围始终受主计划约束。
- **日、周、月视图**：
  - 今日视图按优先级排列任务，完成的计划会自动沉到下方。
  - 本周按每天展示计划，并在当前周自动定位至今天。
  - 本月使用紧凑的表格看板，展示名称、领域、计划日期、优先级和完成度。
- **日历视图**：以月历浏览计划；点击计划打开详情，点击日期可创建计划。
- **领域管理**：内置学习、生活、个人三个领域，也可创建自定义领域或不设置领域。不同领域使用不同颜色标识。
- **优先级**：支持 P1、P2、P3，视图中以醒目颜色区分，高优先级计划优先展示。
- **任务完成度**：无子计划的计划完成后为 100%；含子计划的主计划根据子计划完成情况和占比汇总进度。
- **每周复盘**：查看当前周完成数据、已完成与未完成事项，保存复盘记录；历史保留最近两周并支持通过更多菜单编辑或删除。
- **主题切换**：提供白色、薄荷和深色主题，主题偏好会保存在本地。
- **顺滑交互**：详情、编辑、菜单和视图切换均使用较缓和的过渡动画，避免快速跳变。

## 使用说明

### 1. 登录与数据保存

在登录页输入任意用户名即可开始。iTodo 使用浏览器 `localStorage` 保存账号、计划、领域、主题和复盘记录。清除该浏览器站点数据或更换浏览器后，原数据不会自动恢复。

### 2. 创建计划

1. 通过页面中的 `+` 或“新增计划”打开创建弹窗。
2. 填写计划名称，选择开始与结束日期、优先级及领域。
3. 描述默认折叠，需要时再展开填写。
4. 保存后，计划会出现在其日期范围覆盖的今日、本周、本月和日历视图中。

### 3. 管理计划与子计划

点击任意计划可查看详情并编辑。主计划的详情中可添加子计划；子计划最多嵌套一层，日期会限制在主计划期间内。编辑完子计划后可以返回主计划详情，继续查看其他子计划或修改主计划。

完成计划时，已完成项目会平滑移动到列表下方。月视图中的任务名称可打开详情，日期字段则用于快速查看或调整日期。

### 4. 浏览各个视图

- **今日**：只显示今天覆盖到的计划，空状态可直接点击加号创建。
- **本周**：按天浏览本周计划；切换到其他周时，界面会明确标识非当前周。
- **本月**：以表格形式聚合当月计划，适合比较领域、日期、优先级和完成度。
- **日历**：从月历中快速定位日期和计划。
- **复盘**：以周为单位整理本周成果和遗留事项，并沉淀简短的复盘内容。

### 5. 管理领域和主题

从侧边栏领域区进入管理入口，可新建领域。创建或编辑计划时，领域可选也可留空。主题切换入口位于应用导航中，切换结果会自动记住。

## 技术栈

- React 19
- Vinext / Vite
- Cloudflare Workers 构建适配
- 浏览器 LocalStorage 本地数据存储
