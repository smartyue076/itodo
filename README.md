# iTodo

**把今天要做的事，放进一个真正能推进的计划里。**

iTodo 是一款为个人节奏设计的规划工具。它不要求你先决定这是日计划、周计划还是月计划，只要给一件事一个时间范围，就能在最合适的视图里出现：今天用来执行，本周用来安排，本月用来掌握全局，复盘用来把经验留下来。

## 为什么是 iTodo

### 从项目走到今天

大一点的目标常常卡在“下一步做什么”。iTodo 让你把计划拆成子计划：项目保留整体方向，日与周只呈现需要行动的子计划和独立任务。每次打开，都能看见眼下最值得推进的事。

### 一份计划，多种视角

- **今日**：把注意力留给当下，未完成事项按优先级自然排在前面。
- **本周**：按天展开执行项，随时看见这一周的节奏与空档。
- **本月**：用清晰表格查看项目、类型、日期、优先级和完成度。
- **日历**：从时间维度回看和安排计划，打开时会自动定位当前周。

### 让类型替你说话

工作、学习、生活和个人事项各有自己的颜色。类型和优先级不必写进任务标题，扫一眼就能辨认事项的性质与紧急程度。

### 项目有进度，执行有上下文

子计划可以拥有自己的日期与完成状态，并始终显示自己从属的父计划。主计划会汇总子计划进度，让长期目标不再只是一条没有反馈的清单。

### 每周复盘，也能分享成果

复盘页把已完成、未完成、类型分布和一周节奏放在一起。记录亮点、阻碍和下一步后，可以生成一张简洁的 PNG 总结卡，方便保存或分享。

### 一个舒服的个人工作台

白色、薄荷、深色主题与自定义颜色、显示密度共同组成你的工作环境。界面保持安静，动画保持克制，信息在需要时出现。

## 从第一次打开开始

输入自己的用户名即可开始规划。登录页也提供测试账号，内置一组贴近日常的工作、学习、生活和个人计划，以及可跳过的新手引导，几分钟就能体验完整流程。

## 适合这些时刻

- 你想把一个模糊项目拆成今天真正能做的步骤。
- 你需要同时安排工作、学习与生活，但不想把它们塞进不同工具。
- 你希望每周停下来看看完成了什么、为什么没完成，以及下一步该聚焦哪里。
- 你希望规划内容留在自己手里，随开随用。

## 开始使用

1. 在登录页创建自己的工作区，或选择“使用测试账号”。
2. 新增计划，为它设定日期、类型和优先级。
3. 有长期目标时，在详情中添加子计划。
4. 在今日和本周推进执行项，在本月与日历回看全局。
5. 周末打开复盘页，沉淀这一周的成果与下一步。
